亲属、家庭与收养

专题简目：

亲属、近亲属、家庭成员（了解）

亲属的含义和分类

  1. 亲属包括配偶、血亲和姻亲

    • 配偶不赘述
    • 血亲
      • 直系血亲（蓝色部分）
      • 旁系血亲（绿色部分）
    • 姻亲
      • 血亲的配偶（橙色部分）
      • 配偶的血亲（未画出）

  2. 近亲属（虚线部分）

    • 配偶
    • 父母
    • 子女
    • 兄弟姐妹
    • 祖父母
    • 外祖父母
    • 孙子女
    • 外孙子女

  3. 家庭成员

    • 配偶、父母、子女和其他共同生活的近亲属为家庭成员

近亲属的民法效力

  1. 近亲属之间有法定的抚养义务
  2. 近亲属有相互继承遗产的权利
  3. 近亲属申请宣告失踪、死亡的权利
  4. 一定范围内近亲属禁止结婚（直系血亲和三代以内旁系血亲）
  5. 一定范围内的亲属有法定监护的权利（义务）

收养

收养关系的三方当事人

  1. 被收养人

    • 丧失父母的孤儿
    • 查找不到生父母的未成年人
    • 生父母有特殊困难无力抚养的子女

  2. 送养人

    • 孤儿的监护人
    • 儿童福利机构
    • 有特殊困难无力抚养子女的生父母

  3. 收养人

    • 无子女或者只有一名子女
    • 有抚养、教育和保护被收养人的能力
    • 未患有在医学上认为不应当收养子女的疾病
    • 无不利于被收养人健康成长的违法犯罪记录
    • 年满30周岁

收养的成立于生效要件

  1. 收养关系成立的实质要件

    • 原则上，无子女的收养人可以收养两名子女，有子女的收养人只能收养一名子女；但收养孤儿、残疾未成年人或者儿童福利机构抚养的查找不到生父母的未成年人，可以不受无子女和收养人数规定的限制
    • 无配偶者收养异性子女的，收养人与被收养人的年龄应当相差40周岁以上
    • 无论送养还是收养原则上须经夫妻双方同意
    • 收养、送养双方自愿，且要征得8周岁以上被收养人的同意
    • 未成年人的父母不具备完全民事行为能力且可能严重危害该未成年人的，该未成年人的监护人可以将其送养。监护人送养孤儿的，应当征得有抚养义的人同意。有抚养义务的人不同意送养、监护人不愿继续履行监护职责的，应当按照民法典第一编的规定另行确定监护人

  2. 收养关系成立的形式要件

    • 要式行为：非涉外收养可以签订书面收养协议，也可以不签订；而涉外收养必须签订书面收养协议
    • 国家干预：经过县级以上人民政府民政部门登记；涉外收养的应由省级人民政府民政部门登记；收养查找不到生父母的未成年人的，办理登记的民政部门应在登记前予以公告
    • 收养关系当事人各方或者一方要求办理收养公证的，应当办理收养公证
    • 收养关系成立后，境内收养的办理户口登记

  3. 抚养

    • 祖父母、外祖父母的优先抚养权，优先于收养权
    • 孤儿或者生父母无力抚养，生父母的亲属、朋友抚养的，不适用收养

收养的效力

  1. 收养一经成立，养父母与养子女间的权利义务关系，适用于民法典关于父母子女的规定
  2. 养子女的姓氏有三：随养父、随养母、保留原姓
  3. 收养一经被法院宣告无效，具有溯及力，自始无效

收养协议的解除

  1. 三条途径

    • 收养人、送养人之间的协议解除，此时需要征求8周岁以上被收养人的同意；协议解除后，需要办理解除收养关系登记
    • 送养人单方解除，是指收养人有侵害未成年养子女行为的，送养人有权要求解除收养关系，收养人不同意的，送养人可以起诉解除
    • 养父母与成年养子女关系恶化、无法共同生活的，双方可以协议解除

  2. 解除的效力

    • 养子女与养父母的亲属关系即行消除，与生父母的亲属关系自行恢复，但成年养子女与生父母关系是否恢复，双方自行决定

  3. 养父母有权获得补偿的三种情况

    • 经养父母抚养的成年养子女对缺乏劳动能力又缺乏生活来源的养父母，应当给生活费
    • 因养子女成年后虐待、遗弃养父母而解除收养关系的，养子女补偿收养期间支出的抚养费
    • 生父母单方解除收养关系的，生父母适当补偿收养期间养父母支出的抚养费，单因养父母虐待、遗弃养子女而解除收养关系的除外

家庭关系（了解）

结婚、婚姻效力及离婚

专题简目：

离婚的条件

法定条件

  1. 男女双方完全自愿：禁止一方对另一方的强迫或任何第三人的干涉
  2. 双方年龄到达法定年龄：男满22周岁，女满20周岁
  3. 双方均无配偶
    • 未婚
    • 丧偶
    • 离婚

禁止条件

  1. 未达到法定婚龄的
  2. 非自愿的
  3. 直系血亲或者三代以内旁系血亲
    • 直系血亲不计代数
    • 中国亲等的计算（找出二者共同血缘的源处，以代数最高者为旁系血亲的代数）

结婚登记

登记效力

  1. 我国实行登记婚，经结婚登记，取得结婚证，始确立夫妻关系
  2. 办理结婚登记不得委托代理
  3. 未办理登记的，应该补办登记；补办的，婚姻关系的效力从双方均符合结婚的实质要件时算起，而不是从补办时算起
  4. 离婚后又复婚的，应当重新办理结婚登记

不予登记的情形

  1. 非自愿的
  2. 未达法定婚龄的
  3. 直系血亲或者三代以内旁系血亲的
  4. 已有配偶的

同居关系

  1. 一方或者双方有配偶者与他人同居（非法同居）法律禁止
  2. 双方均无配偶而同居（道德调整）
  3. 当事人起诉请求解除同居关系的，法院原则上不予受理，已受理的，裁定驳回起诉
  4. 无论何种同居关系，当事人因同居期间财产分割或子女抚养纠纷提起诉讼的，法院应予受理

效力瑕疵的婚姻

瑕疵法定主义

无效婚、可撤销婚都实行法定主义。登记程序瑕疵的，不构成效力瑕疵婚，可以寻求行政救济手段

无效婚姻

  1. 无效情形（1051条）

    • 重婚
    • 有禁止结婚的亲属关系
    • 未到法定年龄

  2. 申请宣告人

    • 以重婚为由的，为当事人的近亲属以及基层组织
    • 以未达法定结婚年龄为由的，为未达法定婚龄者的近亲属
    • 以有禁止结婚的亲属关系为由的，为当事人的近亲属

  3. 无效婚姻之诉

    • 当事人仅以一方违法夫妻忠实义务等为由提起诉讼的，法院不予受理；已经受理的，裁定驳回原诉
    • 当事人依1051条向法院请求确认无效的，法定的无效婚姻情形在提起诉讼时已经消失的，不予支持
    • 夫妻一方或者双方死亡后，生存一方或者利害关系人依法请求确认婚姻无效的，法院应予受理
    • 利害关系人为申请人的无效婚姻之诉，婚姻当事人双方均为被申请人；夫妻一方死亡的，生存一方为被申请人；夫妻双方均死亡的，不列被申请人
    • 不调解。法院受理请求确认婚姻无效案件后，原告申请撤诉的，不予准许。对婚姻效力的审理不适用调解，应当依法作出判决。涉及财产分割和子女抚养的，可以调解。调解达成协议的，另行制作调解书；未达成调解协议的，应当一并做出判决
    • 与离婚之诉的关系：法院受理离婚案件后，审查中发现属于无效婚姻的，应作出宣告婚姻无效的判决；法院就同一婚姻关系分别受理了离婚与无效婚姻宣告申请案件的，应先就后者做出判决再进行审理离婚案件；一旦婚姻被宣告无效，应当接着审理涉及财产分割与子女抚养问题

可撤销婚姻

  1. 事由

    • 胁迫
    • 隐瞒重大疾病的欺诈

  2. 撤销权人与撤销机关

    • 只能是受胁迫或者受欺诈一方的婚姻关系当事人本人，撤销机关只能是法院

  3. 除此期间

    • 受胁迫的，1年期间自胁迫结束之日算起；被非法限制人身自由的，自恢复人身自由之日算起
    • 受欺诈的，1年期间自知道或者应当知道受欺诈之日起计算

婚姻被确认无效、被撤销的效力

  1. 具有溯及力。无效的或被撤销的婚姻，自始无效；当事人不具有夫妻权利义务关系，按同居关系处理

  2. 财产处理

    • 当事人在同居期间所得财产，原则上按共同共有处理
    • 有证据证明为一方所有的，归其所有
    • 对于共同共有财产的分配，当事人可以协议处理；协议不成的，按照顾无过错方的原则判决
    • 因重婚导致无效婚姻的，在财产处理上，应准许合法婚姻当事人作为有独立请求权的第三人参加诉讼

  3. 无过错方的损害赔偿请求权

离婚的方式

协议离婚

  1. 协议离婚两个条件

    • 双方自愿
    • 订立书面离婚协议，协议应当裁明双方自愿离婚的意思表示和对子女抚养、财产以及债务处理等事项协商一致的意见

  2. 冷静期

    • 自婚姻登记机关收到离婚登记申请之日起30日内，任何一方不愿意离婚的，可以向婚姻登记机关撤回离婚登记的申请。已经撤回，协议离婚程序即告终止
    • 上述规定期限届满30日内，双方应当亲自到婚姻登记机关申请发给离婚证，才能成功离婚
    • 在上述30日内双方未去申请颁发离婚证的，视为撤回离婚登记申请，协议离婚程序也告终止

  3. 经婚姻登记机关审查，双方确实是自愿离婚，并已经对子女抚养、财产以及债务处理等事项协商一致的，予以登记，发给离婚证

诉讼离婚

  1. 法定情形

    • 重婚或者与他人同居（不忠）
    • 实施家庭暴力或者虐待、遗弃家庭成员（家暴）
    • 有赌博、吸毒等恶习且屡教不改（恶习）
    • 因感情不和而分居满2年（分居）
    • 一方被宣告失踪的（失踪）
    • 法院不判离，双方分居又满1年的，再次起诉离婚的（又分居）

  2. 对两类人的特殊保护

    • 现役军人，为保护军婚，军人的配偶要求离婚的，须征得军人同意；但军人有重大过错的除外
    • 女方，女方孕期；女方分娩后1年内；女方终止妊娠6个月内。男方不得主动提出离婚。（女方提出离婚、法院认为确实有必要受理男方离婚请求的除外）

  3. 必要的程序：调解

亲子关系

亲子鉴定

  1. 认领
  2. 否认
  3. 单向确认：成年子女仅可以向法院提起确认亲子关系之诉而不能提起否认之诉（防止成年子女逃避对养育其长大的父母的赡养义务）

  4. 推定

    • 父或者母向法院起诉请求否认亲子关系，并已提供必要证据予以证明，另一方没有相反证据又拒绝做亲子鉴定的，可以认定否认亲子关系一方的主张成立
    • 父或者母以及成年子女起诉请求确认亲子关系，并提供必要证据予以证明，另一方没有相反证据又拒绝做亲子鉴定的，法院可以认定确认亲子关系一方的主张成立

  5. 人工授精：婚姻存续期间，夫妻双方一致同意进行人工授精，所生子女思维婚生子女

亲权、监护与抚养

  1. 不满2周岁原则上由母亲抚养
  2. 哺乳期后的子女，双方协议；协议不成的由法院依据最有利于未成年子女原则判决
  3. 子女满8周岁的，应尊重其意愿

抚养费

离婚后，一方抚养子女的，另一方应负担必要的生活费和教育费的部分或全部，由双方协议，协议不成的，由法院判决

探望权

离婚的夫妻财产处理问题（见下一专题）

离婚的损害赔偿请求权

责任性质

  1. 物质损害赔偿
  2. 精神损害赔偿

情形

  1. 不忠。重婚或与他人同居的
  2. 家暴。广义上的，包括实施家暴与遗弃、虐待家庭成员
  3. 有其他重大过错，如“同妻”要求男同性恋丈夫赔偿

请求权人与责任人

  1. 请求权人为离婚诉讼当事人中的无过错方
  2. 责任人则是离婚诉讼当事人中无过错方的配偶

请求的条件

  1. 以判决准予离婚为前提
  2. 单方过错为前提
  3. 扩展到协议离婚：当事人在婚姻登记机关办理离婚登记手续后，向法院提出损害赔偿请求的，应当受理；但当事人在协议离婚时已经明确表示放弃该项请求的，不予支持。

请求方式

  1. 无过错方作为原告的，必须在离婚诉讼的同时提出
  2. 无过错方作为被告的，如被告不同意离婚也不提起损害赔偿请求的，可以就此单独提起诉讼
  3. 无过错方作被告的，一审时被告未提起损害赔偿请求，二审期间提出的，法院应调解；调解不成的，告知当事人另行起诉。双方当事人同意由二审法院一并审理的，二审可一并裁判

夫妻财产关系

专题简目：

夫妻财产制

基本原则

  1. 有约定从约定，尊重夫妻双方意思自治
  2. 无约定，法定财产制：婚前财产属于个人；婚内所得财产，属于夫妻共同共有

约定财产制

  1. 对内：夫妻可以自由约定婚前及婚内所得财产的归属。该约定采用书面形式，否则，依旧适用法定财产制。约定一经生效，对双方具有约束力
  2. 对外：夫或妻一方对外所负的个人债务，相对人知道上述夫妻财产约定的，可以对抗该相对人

法定财产制之一：个人财产的认定

下列财产原则上一应为夫或妻一方个人财产，且不因夫妻关系的延续而转化为夫妻共同财产

  1. 一方的婚前财产，永远属于个人所有
  2. 一方因受到人身损害赔偿获得的赔偿和补偿，如医疗费、残疾人生活补助费等费用
  3. 遗嘱或遗赠合同中确定只归一方的财产
  4. 一方专用的生活用品
  5. 夫妻一方个人财产在婚后产生的孳息、自然增值都属于个人财产
  6. 军人的伤亡保险金、伤残补助金、医药生活补助属于个人财产

法定财产制之二：共同共有财产认定的基本原则

在婚姻关系存续期间任何一方所得的财产，原则上属于夫妻共同共有财产

  1. 工资、奖金和其他劳务报酬
  2. 生产、经营、投资的收益
  3. 知识产权的收益
  4. 一方继承、受赠所得财产
    • 对于一方通过法定继承获得的财产，肯定属于夫妻共同共有
    • 对于一方通过遗嘱继承、受遗赠或者受赠与所获得的财产，如遗嘱人、赠与人并未明确表示仅仅归属于一方的，仍然属于夫妻共同共有
    • 对于一方通过遗嘱继承、受遗赠或者赠与所获得的财产，确定只归夫妻一方的财产，应为夫或妻一方财产

法定财产制之三：共同共有财产认定的几处特则

  1. 一方以个人财产投资取得的收益，属于夫妻共有
  2. 男女双方实际取得或者应当取得的住房补贴、住房公积金
  3. 男女双方实际取得或者应当取得的基本养老金、破产安置补偿费
  4. 发送到军人名下的复原费、自主择业费等一次性费用，以夫妻婚姻关系存续年限乘以年平均值，所得数额为夫妻共同财产
  5. 由一方婚前承租、婚后用共同财产购买的房屋

夫妻对共同所有财产的处理权

平等的处理权

  1. 日常生活处理夫妻共同财产的，任何一方均有权决定（家事代理）
  2. 非因日常生活需要对夫妻共同财产做重要处理决定的，双方应平等协商，取得一致意见（否则构成无权处分）
  3. 他人有理由认为前项决定为夫妻双方共同意思表示的，另一方不得以不同意或者不知道为由对抗善意第三人
  4. 夫妻一方未经另一方同意出售夫妻共同所有的房屋，第三人善意购买、支付合理对价并已办理不动产登记，适用善意取得；一方擅自处分共同所有的房屋造成另一方损失，离婚时另一方有权请求赔偿损失

夫妻间的赠与与彩礼处理

  1. 夫妻间的赠与

    • 婚前或婚姻关系存续期间，当事人约定将一方所有的房产赠与另一方或者共有，赠与方在赠与房产变更登记之前撤销赠与，另一方请求判令继续履行的，不允许赠与人任意撤销

  2. 彩礼返还

    • 双方未办理结婚登记手续
    • 双方办理结婚登记手续但确实未共同生活
    • 婚前给付并导致给付人生活困难

离婚诉讼中夫妻财产处理规则

基本原则

  1. 对双方共同财产，双方协议处理
  2. 协议不成的，法院依照照顾子女和女方权益的原则判决
  3. 离婚时，一方有隐藏、转移、变卖、毁损、挥霍共同财产或伪造债务企图侵占另一方财产的情形的，对过错方，法院可以判决其少分或者不分

离婚前分割

婚姻存续期间，有下列情形之一的可以请求分割共有财产

  1. 一方有隐藏、转移、变卖、毁损、挥霍共同财产或伪造夫妻共同债务等严重损害夫妻共同财产利益的行为
  2. 一方负有法定抚养义务的人患重大疾病需要医治，另一方不同意支付相关医疗费用

离婚后的再分割

  1. 离婚后发现一方有隐藏、转移、变卖、毁损、挥霍共同财产或伪造债务企图侵占另一方财产的，或者离婚时尚有未涉及分割的夫妻共同财产的，当事人可诉请法院再次分割财产
  2. 适用普通诉讼时效期间3年，自当事人发现之日起算

夫妻共同债务与个人债务：认定与清偿

基本问题

个人债务与共同债务的区分认定

  1. 个人债务

    • 婚前，原则上夫或妻一方在婚前成立的债务，为个人债务，除非能够证明所负债务用于婚后家庭共同生活
    • 婚内，夫或妻一方由于个人原因所负下的债务。（夫妻一方与第三人恶意串通，虚构债务，以及夫妻一方在从事赌博、吸毒等违法犯罪活动中所负债务，均非夫妻共同债务）

  2. 夫妻共同债务

个人债务与共同债务的偿还

  1. 个人债务的偿还

    • 婚内，对于个人债务，以个人财产份额为限来偿还，不得执行夫妻共同财产的全部
    • 离婚后，对于个人债务，以离婚后个人财产（如分得的财产份额）为限来偿还，不得对原夫妻另一方主张债务偿还

  2. 夫妻共同债务的偿还

    • 婚内，可以强制执行夫妻共同共有财产，以偿还债务
    • 离婚后，由原夫妻二人承担连带责任
      • 外部，如夫妻离婚协议约定其中一人负责偿还的，则该约定对于这对夫妻内部有效，不得对抗债权人
      • 内部，夫或妻一方就共同债务承担连带清偿责任后，可基于合法依据向另一方追偿
      • 此外，夫或妻一方死亡的，生存一方应对婚姻关系存续期间的共同债务承担全部责任
    • 离婚协议或生效判决、裁定、调解书已对夫妻财产分割问题作出处理的，债权人仍有权就夫妻共同债务向男女双方主张权利；一方就夫妻共同债务承担清偿责任后，可以主张由另一方按照离婚协议或者法院的法律文书承担相应责任
    • 夫或妻一方死亡的，生存一方应当对婚姻关系存续期间的夫妻共同债务承担清偿责任
